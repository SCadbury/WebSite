La dulzura del diseño, ahora en tu código.

CadburyStyle es una librería de estilos elegante y moderna, inspirada en la estética del chocolate Cadbury

Diseñada para brindar una experiencia visual rica, profesional y dulce, CadburyStyle ofrece una colección de componentes, utilidades y estilos personalizables para acelerar tu desarrollo frontend con un toque distintivo.

Usa clases simples, aplica colores únicos y crea interfaces memorables desde el primer momento. Sin complicaciones, solo dulzura visual.

Instalación

Para integrar esta librería a tu proyecto, simplemente enlaza el archivo CSS en tu documento HTML. Asegúrate de ubicarlo dentro del <head>.

Puedes descargarlo directamente haciendo click en el botón a continuación:

Descargar librería

Tu <head> deberá contener la siguiente línea para referenciar correctamente el archivo CSS: 

<link rel="stylesheet" href="ruta/a/CadburyStyle.css">

También puedes usar nuestro CDN oficial para enlazar la librería sin descargar archivos locales: 

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/SCadbury/Cadbury_Style@main/docs/CadburyStyle.css">

Una vez enlazado de cualquiera de estas maneras, ya puedes utilizar todas las utilidades, componentes y clases personalizadas que ofrecemos. ¡Disfruta de la dulzura de CadburyStyle en tu proyecto!

Descubrir

¿Sabías que CadburyStyle cuenta con estilos para todo tipo de intereses?

Marcado estructural y buenas prácticas: CadburyStyle está diseñado para aplicarse sobre una amplia variedad de estructuras HTML. La documentación oficial busca no solo mostrarte cómo utilizar la librería, sino también brindarte ejemplos de buenas prácticas en el uso de HTML semántico.

Esto incluye recomendaciones sobre etiquetas correctas para los distintos elementos disponibles, así como estrategias para mejorar la accesibilidad como el uso adecuado.

Explora la documentación completa para descubrir otros elementos personalizables y utilidades integradas.

Podrás encontrar una gran diversidad de contenido y aprender mucho más de esto en el apartado de documentación.

Accesibilidad

La librería ha sido diseñada pensando en la facilidad de lectura, navegación fluida y compatibilidad, esta provee una forma de uso facil, herramientas para llenar de estilo tus paginas web y asi destacar entre todas las demas.

CadburyStyle ha sido creada con principios de accesibilidad en mente para asegurar que cualquier usuario, ofrece una estructura clara, herramientas semánticas y una combinación de colores con contraste suficiente que mejora la experiencia visual.

Algunas de las ventajas que presenta utilizarla:

🗄️Sintaxis facil de comprender y facil de digitar: clases predecibles y fáciles de recordar.

🗄️Colores y contrastes que favorecen la legibilidad.

🗄️Diseño adaptable a distintos tamaños de pantalla: adaptabilidad garantizada en pantallas móviles, tablets y escritorios.

🗄️Evita el uso excesivo de animaciones: todos los efectos visuales son sutiles.
