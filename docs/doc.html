Este es el comienzo de tu aventura con CadburyStyle!

Aquí encontrarás la Documentación necesaria para comenzar tu proyecto hoy!


Tu librería elegante para interfaces dulces y profesionales. Inspirada en la sobriedad del chocolate y la elegancia del morado Cadbury.


Dale un toque delicioso a tus proyectos web con esta exclusiva colección de componentes inspirados en el inconfundible sabor y calidad del chocolate Cadbury. Con una paleta de colores basada en tonos morados profundos, ricos chocolates y acentos dorados, nuestra librería ofrece una experiencia visual tan irresistible como el propio dulce.


Desde botones suaves como un trozo de chocolate hasta tarjetas elegantes que reflejan la suavidad y el lujo de Cadbury, cada elemento ha sido diseñado para añadir ese toque de indulgencia a tu sitio. Los formularios emulan la cremosidad y textura perfecta, mientras que las alertas dinámicas te ofrecen una experiencia interactiva que resalta con el brillo de un chocolate premium.


Nuestra barra de navegación fluye con la misma suavidad que una barra de Cadbury, guiando a tus usuarios de manera sencilla y elegante por tu web. Además, los distintivos (badges) te permiten resaltar los momentos más dulces y especiales de tu plataforma, al igual que el empaque dorado que identifica a Cadbury.


Con contenedores diseñados para mejorar la experiencia de usuario y una cobertura completa de CSS y HTML, la librería Cadbury Style es la elección perfecta para que tu sitio web tenga la misma dulzura y calidad que el mejor chocolate.


Explora lo siguiente en nuestra librería Cadbury Style:

  • Botones: Elegantes, suaves y de colores deliciosos para una interacción suave.
  • Tarjetas: Ricas en detalles y con un estilo sofisticado que recuerda la textura del chocolate.
  • Formularios: Diseñados para ser tan agradables como un trozo de chocolate en tu boca.
  • Alertas: Notificaciones sutiles y atractivas, al estilo Cadbury.
  • Badges: Distintivos que destacan logros y momentos especiales con un toque dorado.

Layout


Containers

Cadbury Style incluye distintos tipos de contenedores adaptables, decorativos y utilitarios.

Container estándar

Este container tiene un ancho máximo que se adapta según el tamaño de pantalla.


Container fluido

Este container ocupa el 100% del ancho de la ventana sin límites.


Container Outline

Este tipo tiene un borde punteado y sombra ligera para destacar secciones.


Container con fondo

Perfecto para secciones destacadas o encabezados con fondo visual.


Container centrado

El contenido está centrado tanto vertical como horizontalmente.


// Ejemplo de código
const saludo = "Hola Cadbury Style";

Container Personalizable

Se adapta como el estándar pero permite modificadores de centrado y separación.

Alineaciones de texto


Clases para justificar, alinear al centro o a los lados tu contenido.


.text-left - Texto alineado a la izquierda

.text-center - Texto centrado

.text-right - Texto alineado a la derecha

.text-justify - Este es un ejemplo de texto justificado. Esta clase se utiliza cuando se desea que los bordes del párrafo estén alineados tanto a la izquierda como a la derecha, proporcionando un aspecto limpio y uniforme.


Ejemplos de uso

<!-- Texto alineado a la izquierda --> <p class="text-left">Texto a la izquierda</p> <!-- Texto centrado --> <p class="text-center">Texto centrado</p> <!-- Texto alineado a la derecha --> <p class="text-right">Texto a la derecha</p> <!-- Texto justificado --> <p class="text-justify">Texto justificado</p>
